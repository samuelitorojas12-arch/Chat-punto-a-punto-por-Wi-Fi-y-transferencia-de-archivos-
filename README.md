# Chat punto a punto por Wi-Fi

Este proyecto consiste en el desarrollo de una aplicación en Python que permite la comunicación entre dos computadoras mediante un chat simple cliente-servidor usando sockets TCP sobre una red Wi-Fi.

## Objetivo

Permitir el envío y recepción de mensajes en tiempo real entre dos equipos conectados a la misma red o mediante hotspot.

## Requisitos

* Python 3.8 o superior
* Conexión Wi-Fi (misma red o hotspot)

## Ejecución

### Servidor

```bash
python server_chat.py
```

### Cliente

```bash
python client_chat.py
```

##  Estructura

* `src/` → Código fuente
* `docs/` → Documentación
* `images/` → Evidencias visuales
* `results/` → Logs y archivos recibidos
